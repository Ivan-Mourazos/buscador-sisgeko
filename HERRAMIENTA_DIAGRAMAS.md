# Herramienta de Diagramas

La herramienta utilizada para visualizar la estructura de la base de datos es compatible con el lenguaje de modelado **es-ldmd** (Spanish Language Data Modeling Design).

- **Formato**: compatible con DBML / es-ldmd.
- **Uso**: Copiar el contenido de `ESTRUCTURA_BD.md` o el bloque generado en el chat y pegarlo en el editor de la herramienta.
